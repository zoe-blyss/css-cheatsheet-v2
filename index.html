CSS Cheat Sheet

margin: auto

centers element horizontally in container

display: flex

sets flexible length on flexible items

flex-direction: row, row-reverse, column, column-reverse

sets the orientation of flex items

justify-content: flex-start, flex-end, center, space-between, space-around, space-evenly

aligns the flexible container's items when the items do not use all available space on the main-axis (horizontally)

align-items: flex-start, center, flex-end, baseline, stretch

specifies the default alignment for items inside the flexible container on cross-axis (vertically)

align-content: flex-start, center, flex-end, stretch, space-between, space-around, initial, inherit

modifies the behavior of the flex-wrap property – aligns flex lines

align-self: flex-start, center, flex-end, stretch, auto, baseline

sets alignment for individual item, used to override align-items

flex-grow: 1, 2, etc., initial, inherit

specifies how much the item will grow relative to the rest of the flexible items inside the same container

flex-shrink: 1, 2, etc., initial, inherit

specifies how the item will shrink relative to the rest of the flexible items inside the same container

flex-basis: 0, 10px, 20px, etc., auto, initial, inherit

specifies the initial length of a flexible item

order: 1, 2, etc., initial, inherit

specifies the order of a flexible item relative to the rest of the flexible items inside the same container

margin: length in px, pt, cm, etc., %, auto, initial, inherit

sets the extra space around an element

padding: length in px, pt, cm, etc., %, auto, initial, inherit

sets the space in between its content and its border

border: border-width, border-style, border-color

shorthand CSS for setting a border between padding and margin of an element

box-sizing: content-box, border-box, initial, inherit

defines how the width and height of an element are calculated – if they added to the total width set or the content will only reach the total width and height given

width: length in px, pt, cm, etc., %, auto, initial, inherit

sets the width of an element

min-width: length in px, pt, cm, etc., %, initial, inherit

defines the minimum width of an element

max-width: none, length in px, pt, cm, etc., %, initial, inherit

defines the maximum width of an element – if content is larger than the maximum width, will automatically change the height of the element

height: length in px, pt, cm, etc., %, auto, initial, inherit

sets the height of an element – does not include padding, borders, or margins

min-height: length in px, pt, cm, etc., %, initial, inherit

defines the minimum height of an element

max-height: none, length in px, pt, cm, etc., %, initial, inherit

defines the maximum height of an element – if content is larger than the maximum height will overflow – how container will handle the overflowing content is defined by the overflow property

absolute units: cm, mm, in, px, pt, pc

units are fixed and a length expressed in any of these will appear as exactly that size – not recommended for screens

relative units: em, ex, ch, rem, vw, vh, vmin, vmax, %

specify a length relative to another length property – recommended for screens

position: static, relative, absolute, fixed, sticky

specifies the type of positioning method used for an element

z-index: auto, 1, 2

specifies the stack order of an element - an element with greater stack order is always in front of an element with a lower stack order. Only works on positioned elements (position: absolute, position: relative, position: fixed, or position: sticky).

display: grid, inline-grid

sets the grid container, which consists of grid items, placed inside columns or rows

grid item

the child element of a grid container

grid-column: 1 / 5, 1 / span 3, 1 / -1

defines on which column/s to place an item

grid-area: 1 / 2 / 5 / 6 , 2 / 1 / span 2 / span 3

shorthand property for the grid-row-start, grid-column-start, grid-row-end and the grid-column-end properties

grid-gap: 20px 50px;

defines the size of the gap between the rows and columns in a grid layout, and is a shorthand property for grid-row-gap and grid-column-gap

autofit

fits the currently available columns into the space by expanding them so that they take up any available space

autofill

fills the row with as many columns as it can fit

minmax(200px, 1fr), (400px, 50%), (30%, 300px)

sets a size range greater than or equal to min and less than or equal to max

repeat(4, 1fr), (4, 250px), (4, 10px 30% 400px)

repeat columns as many times as indicated by 1st value

1fr

a fractional unit - 1 part of the available space
